Taskmaster

This application automates building tasks using a TKInter GUI and Selenium Web automation.

The app is simply a proof-of-concept. Directly automating game actions in against Travian Games Terms of Service, so the application is not available for download.

Oasis Extension

This Chrome extension plugs oasis troop counts into the calculator hosted on travian.kirlloid.ru.

To use, navigate to the world map and open an oasis pop-up. Then, click the extension icon on the toolbar to open a simulation table.

Download and load into developer mode Chrome as an "unpacked extension".
